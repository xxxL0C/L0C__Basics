## Overview

- Unity version : >= 2022.3 LTS
- Scripting Backend : Both (Mono / IL2CPP)
- Required Packages : None (Only use Unity's built-in APIs)

### Namespaces

- Runtime: `XXXL0C.Basics`
- Editor: `XXXL0C.Basics.Editor`

## Installation

### UnityPackage

Download the latest `L0C__Basics_vX-X-X.unitypackage` from the Releases,  
and import it into the Unity Editor.

### Package Manager

copy and paste the URL below in `Add package from git URL` .

```
https://github.com/xxxL0C/L0C__Basics.git?path=Assets/XXXL0C/Basics
```
